# Hexis

A simple, fast habit tracker built with **React + Vite + TypeScript** on the
frontend and **[Convex](https://convex.dev)** for the realtime backend and
database.

## Features in this scaffold

- Create habits with a color and frequency (daily / weekly)
- Toggle a habit complete for the day — synced in realtime via Convex
- Per-habit daily streak counter
- Archive habits you no longer want to track
- Tailwind CSS styling with light/dark support

## Tech stack

| Layer    | Choice                          |
| -------- | ------------------------------- |
| Frontend | React 18, Vite, TypeScript      |
| Backend  | Convex (database + functions)   |
| Styling  | Tailwind CSS                    |
| Tooling  | ESLint, Prettier               |

## Project structure

```
│   ├── main.tsx            # entry point + ConvexProvider wiring
│   └── index.css           # Tailwind entry
├── index.html
├── package.json
├── vite.config.ts
├── tailwind.config.js
└── tsconfig*.json
```

## Getting started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Set up Convex** (creates a deployment and the `convex/_generated` folder,
   and writes `VITE_CONVEX_URL` to `.env.local`):

   ```bash
   npx convex dev
   ```

   Or copy `.env.local.example` to `.env.local` and fill it in manually.

3. **Run the app** (frontend + Convex backend together):

   ```bash
   npm run dev
   ```

   The frontend is served by Vite (default http://localhost:5173) and talks to
   your Convex deployment.

## Scripts

| Command           | Description                              |
| ----------------- | ---------------------------------------- |
| `npm run dev`     | Run Vite + `convex dev` in parallel      |

| `npm run preview` | Preview the production build             |
| `npm run lint`    | Lint with ESLint                         |
